# Linux or Windows from?

- Язык: &nbsp; `python` ;
- Фреймворки: &nbsp; `mlflow`, `airflow` ;
- Библиотеки: &nbsp; `sklearn`, `pyyoutube` , `yaml` , `numpy` , `pandas` , `nltk`, `pymystem3` ;
